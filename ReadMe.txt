This repository contains the main scripts related to: 
Busch, K., Murillo, F.J., Lirette, C., Wang, Z., Kenchington, E. (2024) Putative past, present, and future spatial distributions of deep-sea coral and sponge microbiomes revealed by predictive models.

Input data can be found in the Mendeley database: doi:10.17632/fx3vd2tgcf.1 
if not stated otherwise.

Please take a look at the manuscript above for methodological literature references that are the basis for this code.
